* Security Operations Center (SOC) Investigations
* Digital Forensics
* Incident Response
* Threat Hunting
* Malware Analysis
* Security Monitoring

* Cybersecurity Case Studies

## Objectives

* Demonstrate practical cybersecurity skills.
* Document real-world security investigations and laboratory exercises.
* Maintain a structured collection of cybersecurity knowledge and resources.
* Showcase professional growth and technical competencies.

Contains security monitoring exercises, alert triage activities, log analysis, and SOC-related case studies.

### Digital Forensics

Includes forensic acquisition, evidence handling, disk analysis, memory analysis, and investigation reports.

### Threat Hunting

Documents proactive threat hunting exercises, indicators of compromise (IOCs), and detection methodologies.

### Incident Response

Contains incident response procedures, playbooks, and security incident investigations.

### Security+ Notes

Structured study notes and practical exercises aligned with CompTIA Security+ objectives.

### Malware Analysis

Research notes, malware behavior observations, and analysis reports.

### Case Studies

Cybersecurity scenarios, lessons learned, and security assessments.
